##### How it Wors?
`$cd catkin_ws/src`
`$git clone https://github.com/Kucukcollu/four_dof_manipulator.git`
`$cd catkin_ws/`
`$catkin_make`

###### Simulation on RViz
`$roslaunch four_dof_manipulator display.launch`

###### Simulation on Gazebo
`$roslaunch four_dof_manipulator gazebo.launch`
